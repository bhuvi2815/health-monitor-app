📋 Feature Breakdown

💊 Medicine Alarm

  • User inputs medicine name + time
  • Scheduled local notification
  • Repeats daily automatically
  • Mark as taken / snooze option

🎙️ Voice Assistant

  • Tap mic → speak your question
  • Speech-to-text converts voice
  • OpenAI GPT answers health queries
  • Text-to-speech reads response aloud

📊 Health Monitor

  • Input: BP, pulse, sugar, temperature
  • Live charts with FL Chart library
  • Alert if values exceed safe range
  • History log of all readings

📞 Auto Phone Call

  • If medicine missed → auto trigger
  • Twilio dials the patient's number
  • Recorded voice message plays
  • Works on ANY phone (no smartphone needed)

🛠️ Tech Stack Used

🎯 Flutter (Dart) 🔔 flutter_local_notifications 🎙️ speech_to_text 🔊 flutter_tts 📊 fl_chart 📞 Twilio REST API 🤖 OpenAI GPT API 💾 SharedPreferences 🌐 http package 🔑 permission_handler
